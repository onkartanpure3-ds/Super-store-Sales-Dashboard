# Super-store-Sales-Dashboard
Overview

The Super Store Sales Dashboard is an interactive business intelligence dashboard created using Power BI Desktop. This dashboard helps analyze sales performance, profit trends, customer segments, shipping modes, and payment methods for a superstore business.

The dashboard provides valuable insights through visual representations, enabling better business decision-making and performance tracking.
